# Markdown 문법

## 제목(heading)

제목은 `#`으로 표현한다. `#`의 개수가 늘어날수록 작아진다.

주의사항 `#`과 텍스트 사이에 반드시 띄어쓰기를 넣어야 한다. #제목 <- 붙어있으면 적용 안됨

## 목록

### 순서가 없는 목록

`*` 을 사용한다.

* 순서가 없음
  * tab을 통해서 하위 레벨로 
* shift + tab을 통해서 상위 레벨로 (엔터키를 여러 번 눌러도 가능하다.)



1. 순서가 있는 목록
2. 순서가 있음 
   1. tab을 통해서 하위 레벨로
3. shift + tab을 통해서 상위 레벨로
   * 섞어서 사용도 가능
